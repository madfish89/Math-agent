How It Works

1. Type any math problem.

2. GPT-4o-mini solves it with step-by-step reasoning and proofs.

3. Python (running in your browser via Pyodide + sympy) generates graphs, histograms, and scatter plots when needed.

4. Every answer is double-checked by re-deriving it with Python.

5. You see the answer, steps, confidence score, and graph — nothing else.

Multi-Agent mode: Three agents with different math expertise solve the problem independently, then a synthesis agent combines their answers into one verified solution.
